ECMAScript概述

2021-03-31 JavaScript ECMAScript 0 字数统计: 1.1k(字) 阅读时长: 4(分)

1. ECMA-262

ECMA是一个致力于信息技术标准化的国际组织。ECMA-262，即ECMAScript规范，是它应用最广的规范之一。

ECMA-262第一版诞生于1997年6月，全文110页。到2020年6月发布的第11版，全文已经超过了800页。官方提供了第11版的在线查阅地址、各个ECMAScript版本的PDF下载链接，以及实时更新的最新草案。

下面是ECMA-262的发布历程：

版本 发布时间 主要的新特性
1 1997年6月 数据类型、符号约定、源文本、词法、类型转换、执行上下文、表达式、声明、函数、进程、内置对象、Errors
2 1998年8月
3 1999年12月 Unicode字符集、正则表达式直接量、switch、throw、try、 RegExp对象、Error对象等
4 未公开发布
5 2009年12月 JSON语法和JSON对象、严格模式、debugger语句等
5.1 2011年6月
6 2015年6月 let和const、延展操作符、解构赋值类、模块化、箭头函数、Promise等
7 2016年6月 Array的includes方法、指数运算符**
8 2017年6月 async/await、SharedArrayBuffer对象、Atomics对象等
9 2018年6月 异步迭代、正则表达式扩展、Rest参数语法等
10 2019年6月
11 2020年6月

ECMAScript6（简称ES6）相比上一个大版本ES5相隔近6年，增加包括Class、Promise在内的不少重要特性，具体可查阅阮一峰的ES6入门教程。自ES6之后，ECMAScript开始稳定于每年6月发布一个新版本，每个版本的新特性可参考ES6、ES7、ES8、ES9、ES10新特性一览。

在web开发中，ES新特性的落地需要浏览器的支持，因此在使用新特性时需要注意不同浏览器的兼容问题。

2. ECMAScript2020

ECMAScript2020全文包括27个章节，详情如下：

No 英文名 中文翻译 说明
1 Scope 本文范围
2 Conformance 一致性
3 Normative References 引用资料 ISO/IEC 10646、ECMA-402(API规范)、ECMA-404(JSON)
4 Overview 总览
5 Notational Conventions 符号约定
6 ECMAScript Data Types and Values ECMAScript数据类型与值 七种基本类型与Object引用类型
7 Abstract Operations 抽象操作 类型转换
8 Executable Code and Execution Contexts 可执行代码与执行上下文 词法环境、作用域
9 Ordinary and Exotic Objects Behaviours 普通和非常规对象行为 普通对象的内置方法与slot、函数对象、内置函数对象
10 ECMAScript Language: Source Code ECMAScript语言：源代码
11 ECMAScript Language: Lexical Grammar ECMAScript语言：词法语法 注释、关键字、直接量等
12 ECMAScript Language: Expressions ECMAScript语言：表达式 各种表达式与操作符说明
13 ECMAScript Language: Statements and Declarations ECMAScript语言：语句与声明 if、continue、break、return、with、switch、throw、try、debugger
14 ECMAScript Language: Functions and Classes ECMAScript语言：函数与类 箭头函数、方法、Generator、Class、异步函数
15 ECMAScript Language: Scripts and Modules ECMAScript语言：脚本与模块
16 Error Handling and Language Extensions 错误处理与语言扩展
17 ECMAScript Standard Built-in Objects ECMAScript标准内置对象
18 The Global Object 全局对象 全局对象的值、函数、构造属性及其他
19 Fundamental Objects 基础的对象 狭义对象、Function对象、Boolean对象、Symbol对象、Error对象
20 Numbers and Dates 数值与日期 Number对象、BigInt对象、Math对象、Date对象
21 Text Processing 文本处理 String对象、RegExp对象
22 Indexed Collections 索引集合 Array对象、TypedArray对象
23 Keyed Collections 键集合 Map对象、Set对象、WeakMap对象、WeakSet对象
24 Structured Data 结构型数据 ArrayBuffer对象、SharedArrayBuffer对象、DataView对象、Atomics对象、JSON对象
25 Control Abstraction Objects 控制抽象对象 迭代、GeneratorFunction对象、AsyncGeneratorFunction对象、Generator对象、AsyncGenerator对象、Promise对象、AsyncFunction对象
26 Reflection 反射 Reflect对象、Proxy对象
27 Memory Model 内存模型

除了1-4章，剩下的章节组织结构如下（见4.4章）：

  1. 第 5 章定义了贯穿整个规范的符号约定
  2. 第 6-9 章定义了ECMAScript进程操作的执行环境
  3. 第 10-16 章定义了具体的ECMAScript进程语言，包括它所有语言特征的句法编码与执行语义。
  4. 第 17-26 章定义了ECMAScript标准库，包括所有标准对象的定义，一旦ECMAScript进程开始执行，这些对象就是可用的。
  5. 第 27 章定义了支持访问SharedArrayBuffer内存的内存一致性模型，以及原子对象的方法。
